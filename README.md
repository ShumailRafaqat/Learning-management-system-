Description:
This website curates courses from various tutors and organizes them with playlists to streamline the learning process. It provides an easy and organized way for learners to discover and follow courses that match their interests.

Technologies Used:

HTML
CSS
JavaScript
Features:

Course Curation: Browse a collection of courses from different tutors.
Playlists: Access organized playlists for each course.
User-Friendly Interface: Simple and intuitive navigation to enhance the learning experience.
Installation:

Clone the repository:
git clone [repository-url]
Navigate to the project directory:
cd [project-directory]
Open the index.html file in your web browser.
Usage:

Visit the website to browse through curated courses and explore playlists.
Use the navigation menu to find different tutors and courses.
